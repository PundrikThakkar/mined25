# PLAY-PULSE

Welcome to **PLAY-PULSE repo**! This is an AI powered web app that allows you to create stunning montages of your recorded Valorant gameplay videos. Using our trained YOLO model, we detect highlights in your video and generate a montage. You can also upload an optional music file to be included in the montage.

## Features

- **Upload Gameplay Videos**: Easily upload your recorded Valorant gameplay videos.
- **Optional Music**: Add a music file in mp3 or mpeg format to enhance your montage.
- **Highlight Detection**: Our YOLO model detects the best moments in your video.
- **Download Montages**: Get a downloadable montage of your highlights.

## Getting Started

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>


2. Navigate to the frontend directory and install the dependencies:
    ```sh
    cd frontend
    npm install

3. Navigate to the backend directory and install the dependencies:
    ```sh
    cd ../backend
    npm install


### Running the Project

1. Start the frontend:
    ```sh
    cd frontend
    npm run dev

2. Start the backend:
    ```sh
    cd backend
    npm run dev

### Usage
1. Open your browser and navigate to http://localhost:5173.
2. Upload your recorded Valorant gameplay video.
3. Optionally, upload a music file in mp3 or mpeg format.
4. Click the "Upload" button to generate and download the montage.
